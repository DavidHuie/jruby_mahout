# Jruby Mahout
Jruby Mahout is a gem that unleashes the power of Apache Mahout in the world of Jruby.

## Quick Overview
Apache Mahout is a machine learning library developed for Java.

## Installation
### Add the gem to your `Gemfile`
```ruby
platform :jruby do
  gem "jruby_mahout"
end
```
And run `bundle install`.
